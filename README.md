# INTRODUCTION

Prompting is the process of giving instructions or questions to an AI model to generate responses. The quality of the output depends greatly on how clearly the prompt is written.

There are two important prompt styles:

### 1. Naïve Prompt

A naïve prompt is short, vague, or unstructured. It gives very little context or guidance to the AI model.

**Example:**
“Tell me about health.”

### 2. Basic Prompt

A basic prompt is clear, refined, and structured. It provides context, instructions, and expected output format.

**Example:**
“Explain the importance of maintaining good health through balanced diet, exercise, and proper sleep with suitable examples.”

Basic prompts usually generate more accurate, detailed, and useful responses.
